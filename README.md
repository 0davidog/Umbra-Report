## Agile Process

### Initial User Stories

#### Iteration 01 – Due 05/03/2024

As this is an accedemic porfolio project there is only one interation in which this project will be developed and tested. That being the project start date to it's hand-in date by which all its must-have deliverables must be achieved.

**Epic: User Registration and Authentication** 

- As a Site User, I want to register an account on the blog website so that I can create and manage my own posts. MH 
- As a Site User, I want to log in to the website securely using my credentials so that I can participate in the blog’s features. MH 
- As a Site user, I want the option to reset my password in case I forget it, ensuring secure access to my account. SH 

**Epic: Blog Post CRUD Management** 

- As a Site User, I want to create a new blog post with a title, content, and optional images, so that I can share my thoughts and experiences. MH
- As a Site User, I want the ability to update my existing blog posts to update or improve the content over time. MH
- As a Site User, I want to be able to delete my own blog posts if I decide to remove them from the website. MH 
- As a Site User or Admin, I can create draft posts so that I can finish writing the content later. SH
- As a Site Admin I can manage all blog posts through the admin panel to maintain control over the site’s content. SH

 **Epic: User Interaction and Engagement** 

- As a Site User I can view a paginated list of posts so that I can select which post I’m interested in viewing SH 
- As a Site User, I can click on a post so that I can view the complete content. SH 
- As a Site User or Admin, I can view the comments on each individual post so that I can read the conversation 
- As a Site User, I want to be able to comment on blog posts to engage with the author and other readers. SH 
- As a Site Admin I can approve or disapprove comments so that I can filter out objectionable comments. 
- As a Site User, I want to be able to like blog posts to engage with the author and other readers. SH
- As a Site User, I want to receive notifications when someone likes or comments on my blog posts to stay informed about the engagement with my content. WH 

**Epic: Search and Navigation** 

- As a Site User, I want to search for specific blog posts or topics to quickly find the information I'm looking for. CH
- As a Site User, I want a user-friendly navigation menu that helps me explore different sections of the blog easily. CH 

**Epic: Responsive Design and Accessibility** 

- As a Site User, I want the blog website to be responsive, ensuring a seamless experience on various devices, including desktops, tablets, and smartphones. SH 
- As a Site User with accessibility needs, I want the website to be accessible, with features like alt text for images and keyboard navigation, to ensure a positive experience for all users. SH 

**Epic: Additional information** 

- As a Site User, I can click on the About link so that I can read about the site. 
- As a Site Admin, I can create or update the about page content so that it is available on the site.
